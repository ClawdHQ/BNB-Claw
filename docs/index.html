Complete architecture for agent-driven DeFi

BNB Claw separates concerns cleanly so teams can extend behavior safely while keeping execution deterministic and observable.

  • Agent Layer: intent parsing, reasoning, policy checks, and action planning.
  • Execution Layer: module invocation, retries, simulation, signing, and transaction handling.
  • Protocol Modules: Swap, Lend, Stake, and Treasury out of the box.
  • Controls Layer: spend limits, risk thresholds, route allowlists, role permissions.
  • Telemetry Layer: logs, action history, failure classification, and audit trails.
quickstart.ts 
import { BaseAgent } from "@bnbclaw/core";
import { SwapModule, LendModule, StakeModule } from "@bnbclaw/modules";

const agent = new BaseAgent({
  name: "TreasuryOperator",
  chain: "bsc-mainnet",
  safety: {
    maxTradeUsd: 25000,
    allowedTokens: ["BNB", "USDT", "BTCB", "ETH"],
    requireSimulation: true
  }
});

agent.registerModule(new SwapModule());
agent.registerModule(new LendModule());
agent.registerModule(new StakeModule());

await agent.execute("Shift 12% idle BNB to lowest-risk yield venue");
