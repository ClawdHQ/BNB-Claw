Quick Start

From zero to autonomous DeFi agent in under 5 minutes.

1

Install packages

terminal 
npm install @bnb-claw/core @bnb-claw/modules
2

Create an agent

agent.ts 
import { BaseAgent } from '@bnb-claw/core';
import { SwapModule, LendModule } from '@bnb-claw/modules';
import { ethers } from 'ethers';

const provider = new ethers.JsonRpcProvider('https://bsc-dataseed.binance.org');
const signer   = new ethers.Wallet(process.env.PRIVATE_KEY!, provider);

const agent = new BaseAgent({
  name: 'DeFiBot',
  description: 'Autonomous BNB Chain DeFi agent',
  model: 'gpt-4',
  provider: 'openai',
});
3

Register modules

agent.ts 
const swapModule = new SwapModule({
  name: 'swap', description: 'PancakeSwap V3', version: '0.1.0',
  chainId: 56, provider, signer,
  routerAddress: '0x13f4EA83D0bd40E75C8222255bc855a974568Dd4',
  quoterAddress: '0xB048Bbc1Ee6b733FFfCFb9e9CeF7375518e25997',
  factoryAddress: '0x0BFbCF9fa4f9C56B0F40a671Ad40E0805A091865',
  defaultSlippage: 50, // 0.5%
});

agent.registerModule('swap', swapModule);
4

Execute DeFi actions with natural language

agent.ts 
// Plan and execute from natural language
const response = await agent.reason('Swap 100 USDT to BNB');

// Or execute a specific action directly
const result = await agent.execute({
  type: 'swap',
  module: 'swap',
  method: 'swap',
  params: {
    tokenIn:  '0x55d398326f99059fF775485246999027B3197955', // USDT
    tokenOut: '0xbb4CdB9CBd36B01bD1cBaEBF2De08d9173bc095c', // WBNB
    amountIn: '100000000000000000000', // 100 USDT (18 decimals)
  },
  reasoning: 'User requested USDT→BNB swap',
});

console.log('Tx hash:', result.txHash);

API Reference

Core classes and methods at a glance.

BaseAgent

MethodSignatureDescription
registerModule (name: string, module: any): void Register a DeFi module with the agent.
reason (input: string, ctx?: Partial<AgentContext>): Promise<AgentResponse> Pass natural language to the LLM and get back structured actions.
execute (action: AgentAction, onProgress?): Promise<any> Execute a single action with automatic retry and backoff.
planStrategy (goal: string, ctx?: Partial<AgentContext>): Promise<AgentAction[]> Ask the LLM to produce a multi-step action plan.
chat (message: string): Promise<string> Simple conversational interface; returns the LLM's text reply.
clearHistory (): void Reset the conversation history stored in the agent's context.

SwapModule

MethodReturnsDescription

checkPriceImpact(params)Promise<number>Return price impact percentage for a prospective swap.
getRoute(tokenIn, tokenOut, amount)Promise<Route>Find the optimal swap route.

repay(params)Promise<RepayResult>Repay a borrow position.

FAQ

Answers to common questions.

Which networks are supported?

BNB Chain mainnet (Chain ID 56) and BSC Testnet (Chain ID 97) are supported out of the box. The SDK is extensible — you can configure custom providers for other EVM-compatible chains.

Do I need an OpenAI API key?

Yes, if you're using the openai provider. Set OPENAI_API_KEY in your environment or pass it via the apiKey config option. Anthropic Claude is also supported via ANTHROPIC_API_KEY.

Is BNB Claw safe for production?

The SDK implements price-impact checks, input validation, retry logic, and sensible defaults. Always test on testnet first, verify actions before submitting them to mainnet, and use the needsApproval flag for high-risk operations.

Can I build custom modules?

Yes. Extend the Module base class from @bnb-claw/core, implement getMetadata(), and register your module with agent.registerModule('myModule', instance).

What version of ethers.js is used?

BNB Claw uses ethers.js v6. Make sure your project uses the same major version to avoid type conflicts.

Is the SDK open source?

Yes! BNB Claw is MIT licensed. Find the source on GitHub.
